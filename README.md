# คันจิ แฟลชการ์ด N5 🇯🇵

แฟลชการ์ดคำศัพท์/คันจิภาษาญี่ปุ่นระดับ **JLPT N5** สำหรับคนไทย — คำอธิบายและตัวอย่างประโยคเป็น**ภาษาไทย** ปัดการ์ดแบบ Tinder มีเสียงอ่าน และ**ติดตั้งบนมือถือใช้ออฟไลน์ได้** (PWA)

**เปิดใช้งาน:** [เปิดแอพ](https://ball2b-cpu.github.io/kanji-n5/)

---

## ✨ ฟีเจอร์
- **928 คำ** — N5 ครบ 849 คำ + N4 เริ่มต้น 79 คำ (คัดและ cross-check กับพจนานุกรมแล้ว)
- อธิบาย + ตัวอย่างประโยค **ภาษาไทย** ทุกคำ (ไม่ใช่อังกฤษ)
- **ปัดการ์ด** ซ้าย/ขวา + เฉลยสองจังหวะ (กดดูคำแปลก่อนให้คะแนน)
- **เสียงอ่าน** คำและประโยค (ญี่ปุ่น) ผ่านเสียงในเครื่อง
- ระบบทบทวนอัจฉริยะ **Leitner SRS** — จำคำที่ยังไม่แม่นถี่ขึ้น
- นับ **streak** และเก็บความคืบหน้าในเครื่อง (localStorage)
- **ติดตั้งเป็นแอพ** บน iPhone/Android → ใช้ออฟไลน์ได้เต็มจอ

## 📱 วิธีติดตั้งบน iPhone
1. เปิดลิงก์แอพด้วย **Safari**
2. แตะปุ่ม **แชร์** (􀈂) ล่างจอ
3. เลือก **"เพิ่มไปยังหน้าจอโฮม" / Add to Home Screen** → **เพิ่ม**
4. เปิดครั้งแรกตอนมีเน็ต ปัดสัก 2–3 ใบ → หลังจากนั้น**ใช้ออฟไลน์ได้ตลอด**

> Android: เปิดด้วย Chrome → เมนู ⋮ → "ติดตั้งแอป / Add to Home screen"

## 🔤 ที่มาข้อมูล
- **คำอ่าน (reading/romaji):** สร้างด้วย [fugashi](https://github.com/polm/fugashi) (MeCab + unidic-lite) และ [cutlet](https://github.com/polm/cutlet) แล้ว cross-check กับพจนานุกรม **EDICT2 (EDRDG)**
- **คำแปลไทย + ตัวอย่างประโยค:** เรียบเรียงด้วย AI แล้วตรวจแก้ orthography คันจิด้วยมือ
- ลิสต์คำ N5 อ้างอิงลิสต์มาตรฐานที่ใช้กันแพร่หลาย
